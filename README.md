# UK Supplier Payment Behaviour Forecast Dashboard

An interactive Streamlit dashboard for exploring and forecasting UK firm level supplier payment behaviour, built on the merged Prompt Payment Reporting (PPR) and FAME analytical panel developed as part of an MSc dissertation at UCL.

## What the dashboard does

The dashboard surfaces four views of UK non-financial firms' supplier payment behaviour over the period 2017 to 2025:

- **Firm explorer.** Search for a firm by name or CRN and inspect its historical payment behaviour, financial trajectory, and position relative to its sector.
- **Forecast.** One year and three year ahead forecasts of firm level payment behaviour, generated by two independent models (a panel autoregression with firm level controls, and an XGBoost gradient-boosted tree model) with comparison and confidence intervals.
- **Sector dashboard.** Aggregate forecasts and current behaviour across the 17 UK SIC sectors, with the flagship Manufacturing and Administrative & Support Services sectors highlighted.
- **Model validation.** Held out backtest results showing the two forecast models' accuracy on 2024 data, by sector and overall.

## Methodology

The dashboard operationalises the analytical panel developed for the dissertation, comprising 8,370 firm year observations across 1,390 unique non financial firms over 2017 to 2025. Two forecasting models are implemented and compared:

- **Panel autoregression.** A firm fixed-effects model with one year lagged payment time and firm characteristics (capital intensity, log total assets, profit margin, leverage, working capital intensity) as predictors.
- **XGBoost gradient boosting.** A non parametric model using the same features plus engineered lags (1, 2, and 3 years) and sector indicators.

Both models are trained on 2017 to 2023 data and evaluated against held-out 2024 observations. Forecast accuracy is reported using RMSE and MAPE.

## Running locally

Requires Python 3.13 or later. From the repository root:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app/main.py
```

The dashboard will open in your browser at `http://localhost:8501`.
